The Countries resource lists countries in the OpenAQ dataset and details the
air quality parameters for each. The list only includes countries with
monitoring locations from which OpenAQ ingests data. Country values come from
the coordinates provided by the upstream provider. OpenAQ uses the
[Natural Earth 10m dataset](https://www.naturalearthdata.com/) for country
boundaries. Country codes follow the
[ISO 3166-1 alpha-2](https://www.iso.org/iso-3166-country-codes.html)
standard, which may differ from codes reported by the upstream provider.

## Methods

### Get a single country
```py
get(
  countries_id: int
  ) -> CountriesResponse
```

| argument | type |
| -------- | ---- |
| countries_id | `int` |

#### Example

<Tabs syncKey="client">
  <TabItem label="sync">
```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.countries.get(countries_id=1)
```
  </TabItem>
  <TabItem label="async">
```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:
        await client.countries.get(countries_id=1)
```
  </TabItem>
</Tabs>

### List countries
```py
list(
  page: int = 1,
  limit: int = 1000,
  order_by: str | None = None,
  sort_order: SortOrder | None = None,
  parameters_id: int | list[int] | None = None,
  providers_id: int | list[int] | None = None
  ) -> CountriesResponse
```

| argument | description | input requirements |
| -------- | ----------- | ------------------ |
| page | `int` The page number to retrieve, defaults to 1 | Must better greater than zero |
| limit | `int` The number of results returned per page, defaults to 1,000 | Must range from 1 to 1,000 |
| order_by | `str \| None` The attribute by which to order results, defaults to `id` | Currently only `id` is supported |
| sort_order | `SortOrder \| None` The order by which to sort the attribute, defaults to `desc` | Must be one of: `asc`, `desc`, `ASC`, `DESC` |
| parameters_id | `int \| list[int] \| None` Filter countries by the ID of one or multiple of pollutants or meteorological parameters |  |
| providers_id | `int \| list[int] \| None` Filter countries by the ID of one or multiple data providers | |

#### Example

<Tabs syncKey="client">
  <TabItem label="sync">
```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.countries.list()
```
  </TabItem>
  <TabItem label="async">
```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:
        await client.countries.list()
```
  </TabItem>
</Tabs>

## Response
```
results[]
    ├── id: int
    ├── code: str
    ├── name: str
    ├── datetime_first: 
    │   ├── utc: str
    │   └── local: str
    ├── datetime_last: 
    │   ├── utc: str
    │   └── local: str
    ├── parameters: []
    │   ├── id: int
    │   ├── name: str
    │   ├── units: str
    │   ├── display_name: str | None
    │   └── parameter_type: str
    └── providers: []
        ├── id: int
        └── name: str
```
