### List multiple countries

```py
list(
  page: int = 1,
  limit: int = 1000,
  order_by: str | None = None,
  sort_order: SortOrder | None = None,
  parameters_id: int | list[int] | None = None,
  providers_id: int | list[int] | None = None
  ) -> CountriesResponse
```

| argument | type | description |
| -------- | ---- | ----------- |
| page | `int` | The page number, must be greater than zero. Page count is countries found / limit. |
| limit | `int` | The number of results returned per page. Must be between 1 and 1,000. |
| order_by | `str \| None` | Order by operators for results. |
| sort_order | `SortOrder \| None` | Order for sorting results (asc/desc). |
| parameters_id | `int \| list[int] \| None` | Single parameters ID or an array of IDs. |
| providers_id | `int \| list[int] \| None` | Single providers ID or an array of IDs. |

#### Example

<Tabs syncKey="client">
  <TabItem label="sync">
    ```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.countries.list()
    ```
  </TabItem>
  <TabItem label="async">
    ```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:
        await client.countries.list()
    ```
  </TabItem>
</Tabs>
