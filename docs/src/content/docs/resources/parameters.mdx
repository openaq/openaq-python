### List multiple parameters

```py
list(
  page: int = 1,
  limit: int = 1000,
  order_by: str | None = None,
  sort_order: SortOrder | None = None,
  parameter_type: ParameterType | None = None,
  coordinates: tuple[float, float] | None = None,
  radius: int | None = None,
  bbox: tuple[float, float, float, float] | None = None,
  iso: str | None = None,
  countries_id: int | list[int] | None = None
  ) -> ParametersResponse
```

| argument | type | description | rules |
| -------- | ---- | ----------- | ----- |
| page | `int` | The page number, must be greater than zero. Page count is parameters found / limit. | |
| limit | `int` | The number of results returned per page. Must be between 1 and 1,000. | |
| order_by | `str \| None` | Order by operators for results. | |
| sort_order | `SortOrder \| None` | Order for sorting results (asc/desc). | |
| parameter_type | `ParameterType \| None` | pollutant or meteorological. | |
| coordinates | `tuple[float, float] \| None` | WGS 84 coordinate pair in form latitude, longitude (y,x). | Must be accompanied by radius, cannot be accompanied by bbox. |
| radius | `int \| None` | A distance value in meters to search around the given coordinates value. | Must be accompanied by coordinates, cannot be accompanied by bbox. |
| bbox | `tuple[float, float, float, float] \| None` | Geospatial bounding box of min X, min Y, max X, max Y in WGS 84 coordinates. Limited to four decimals precision. | Cannot be accompanied by either coordinate or radius. |
| iso | `str \| None` | 2 letter ISO 3166-alpha-2 country code. | |
| countries_id | `int \| list[int] \| None` | Single countries ID or an array of IDs. | |

#### Example

<Tabs syncKey="client">
  <TabItem label="sync">

    ```py
        client.parameters.list()
    ```

  </TabItem>
  <TabItem label="async">

    ```py
        await client.parameters.list()
    ```
  </TabItem>
</Tabs>
