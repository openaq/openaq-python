### Get a single manufacturer

```py
get(
  manufacturers_id: int
  ) -> ManufacturersResponse
```

| argument | type |
| -------- | ---- |
| manufacturers_id | `int` |

#### Example

<Tabs syncKey="client">

  </TabItem>
  <TabItem label="async">

### List multiple manufacturers

```py
list(
  page: int = 1,
  limit: int = 1000,
  order_by: str | None = None,
  sort_order: SortOrder | None = None
  ) -> ManufacturersResponse
```

| argument | type | description |
| -------- | ---- | ----------- |
| page | `int` | The page number, must be greater than zero. Page count is manufacturers found / limit. |
| limit | `int` | The number of results returned per page. Must be between 1 and 1,000. |
| order_by | `str \| None` | Order by operators for results. |
| sort_order | `SortOrder \| None` | Order for sorting results (asc/desc). |

#### Example

<Tabs syncKey="client">
  <TabItem label="sync">
    ```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.manufacturers.list()
    ```
  </TabItem>
  <TabItem label="async">
    ```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:
        await client.manufacturers.list()
    ```
  </TabItem>
</Tabs>
