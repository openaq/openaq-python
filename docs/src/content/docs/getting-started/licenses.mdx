### List multiple licenses

```py
list(
  page: int = 1,
  limit: int = 1000,
  order_by: str | None = None,
  sort_order: SortOrder | None = None
  ) -> LicensesResponse
```

| argument | type | description |
| -------- | ---- | ----------- |
| page | `int` | The page number, must be greater than zero. Page count is licenses found / limit. |
| limit | `int` | The number of results returned per page. Must be between 1 and 1,000. |
| order_by | `str \| None` | Order by operators for results. |
| sort_order | `SortOrder \| None` | Order for sorting results (asc/desc). |

#### Example

<Tabs syncKey="client">
  <TabItem label="sync">
    ```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.licenses.list()
    ```
  </TabItem>
  <TabItem label="async">
    ```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:
        await client.licenses.list()
    ```
  </TabItem>
</Tabs>
