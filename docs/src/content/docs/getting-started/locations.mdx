## Methods

### get a single location


<Tabs  syncKey="client">
  <TabItem label="sync">
    ```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.locations.get()
    ```
  </TabItem>
  <TabItem label="async">
    ```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:
        await client.locations.get()
    ```
  </TabItem>
</Tabs>
