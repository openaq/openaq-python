### Get a single location

```py
get(
  locations_id: int
  ) -> LocationsResponse
```

| argument | type |
| -------- | ---- |
| locations_id | `int` |

#### Example


<Tabs  syncKey="client">
  <TabItem label="sync">
    ```py
    from openaq import OpenAQ

    with OpenAQ() as client:
        client.locations.get()
    ```
  </TabItem>
  <TabItem label="async">
    ```py
    from openaq import AsyncOpenAQ

    async with AsyncOpenAQ() as client:

    ```

###  List multiple locations

```py
list(
  page: int = 1,
  limit: int = 100,
  radius: int | None = None,
  coordinates: tuple[float, float] | None = None,
  bbox: tuple[float, float, float, float] | None = None,
  providers_id: int | list[int] | None = None,
  countries_id: int | list[int] | None = None,
  parameters_id: int | list[int] | None = None,
  licenses_id: int | list[int] | None = None,
  iso: str | None = None,
  monitor: bool | None = None,
  mobile: bool | None = None,
  order_by: str | None = None,
  sort_order: SortOrder | None = None
  ) -> LocationsResponse
```

| argument | description | rules |
| -------- | ---- | ---- |
| page | `int` | |
| limit | `int` | |
| radius | `int \| None` | Must be accompanied by coordinates, cannot be accompanied by bbox.  |
| coordinates | `tuple[float, float] \| None` | Must be accompanied by radius, cannot be accompanied by bbox.  |
| bbox | `tuple[float, float, float, float] \| None` | Cannot be accompanied by either coordinate or radius |
| providers_id | `int \| list[int] \| None` | |
| countries_id | `int \| list[int] \| None` | |
| parameters_id | `int \| list[int] \| None` | |
| licenses_id | `int \| list[int] \| None` | |
| iso | `str \| None` | |
| monitor | `bool \| None` | |
| mobile | `bool \| None` | |
| order_by | `str \| None` | |
| sort_order | `SortOrder \| None` | |

## Response

```
Location[]
    ├── id: int
    ├── name: str
    ├── locality: str | None
    ├── timezone: str
    ├── country: 
    │   ├── id: int
    │   ├── code: str
    │   └── name: str
    ├── owner: 
    │   ├── id: int
    │   └── name: str
    ├── provider: 
    │   ├── id: int
    │   └── name: str
    ├── is_mobile: bool
    ├── is_monitor: bool
    ├── instruments: []
    │    ├── id: int
    │    └── name: str
    ├── sensors: []
    │   ├── id: int
    │   ├── name: str
    │   └── parameter:
    │       ├── id: int
    │       ├── name: str
    │       ├── units: str
    │       └── display_name: str | None
    ├── coordinates: 
    │   ├── latitude: float
    │   └── longitude: float
    ├── bounds: tuple[float, float, float, float]
    ├── distance: float | None
    ├── datetime_first: 
    │   ├── utc: str
    │   └── local: str
    └── datetime_last: 
        ├── utc: str
        └── local: str
```
