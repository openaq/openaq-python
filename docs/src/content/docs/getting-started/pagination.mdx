title: Pagination
---


The OpenAQ platform contains a very large amount of data and the OpenAQ API 
can only return a subset of data in a single requests. The API uses pagination 
to allow querying large result sets across multiple queries. 

To access large amounts of data the OpenAQ Python SDK provides the `page` 
and `limit` arguments for all resources that use a `list()` method.
